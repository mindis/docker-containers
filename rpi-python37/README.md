# Purpose

The purpose of this container is to create a Python 3.7 environment for use with Raspberry Pi 4 (armv7l).

# Docker Hub

[Image](https://hub.docker.com/r/vangjee/rpi-python37)

# Docker

Build it.

```bash
./build.sh
```

Run it.

```bash
docker run -it rpi-python37:local
```
